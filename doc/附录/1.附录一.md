## StartupHelper private constructor()

- 构造函数参数
- 改造函数私有


```java
public class StartupHelper {
    private StartupHelper() {}

    public static final StartupHelper INSTANCE = new StartupHelper();
}
```

等同于

```kotlin
class StartupHelper private constructor() {
    companion object {
        val INSTANCE = StartupHelper()
    }
}
```
