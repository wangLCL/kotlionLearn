# Kotlin开发初体验

目标：

- 开发第一个应用
- 编写代码
- 展示结果

## 开发环境

idea

编写面向对象语言，运行在jvm上的kotlin代码。

也需要jdk，kotlin代码编译为字节码。


## 第一个代码

创建项目就可以书写第一个代码了。

```kotlin
package kw.test
//函数声明  
fun main(args: Array<String>) {
    println("Hello World!")
}
```

main函数有特殊的含义，所有的都是从这个地方开始的。

kotlinc-jvm编译kotlin代码，将kotlin代码转换为jvm语言字节码，将字节码在jvm中运行。


kotlin不仅仅是jvm，也可以是js,也可以编译为原生的二进制代码。
