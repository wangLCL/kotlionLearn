# 第十二章 类

类是核心

## 定义一个类

类可以定义在一个独立的文件中，也可以和函数或者变量定义在一起。

类在单独的文件中可以给升级预留空间。

类名和文件名一样，也可以不一样  一个文件中可以定义多个类

```kotlin
class Player
```

### 定义构造

创建实体，必须实例化类，调用它的构造函数

```kotlin
var player = Player()
```
