# 面向对象

Kotlin语言目前还是以面向对象编程为主，函数式编程为辅。面向对象是Kotlin是重要
的特性之一。

## 声明

```kotlin
class 类名{
    类的成员
    
}
```

类成员包括：
- 构造函数
- 初始化代码
- 成员函数
- 属性
- 内部类很嵌套
- 对象表达式声明

```kotlin

class Animal{}
or 
class Animal

```

### 声明属性

```kotlin

class Animal{
    var age = 1
    var sex = false
    private val weight = 0.0
}
```

### 声明成员函数


```kotlin

class Animal{
    var age = 1
    var sex = false
    private val weight = 0.0
    
    private fun eat(){
        
    }
    
    fun run():Int{
        return 10
    }
    
    fun getMaxNumber(n1:Int,n2:Int) = if (n1>n2)n1 else n2
}
```
